# Programming Fundamentals Practice

This repository contains my **practice exercises for Programming Fundamentals (PF)** using C++.  
The goal is to strengthen my understanding of basic programming concepts and problem-solving skills.

## Topics Covered

- **Variables & Data Types** – Basics of C++ data types and variables.  
- **Operators & Expressions** – Arithmetic, relational, logical, and assignment operators.  
- **Control Flow** – `if`, `else`, `switch`, loops (`for`, `while`, `do-while`).  
- **Functions** – Function declaration, definition, recursion, and overloading.  
- **Arrays & Strings** – One-dimensional and multi-dimensional arrays, string manipulation.  
- **Pointers & References** – Basics of pointers and references in C++.  
- **Simple Projects** – Mini practice programs applying the above concepts.
